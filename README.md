# Proyecto P3_Aitana DevOps
Este repositorio contiene un proyecto de JavaScript de 4 ejercicios hecho en clase.
## Estructura del proyecto
- css: Dentro se encuentra la hoja de estilos y una tipográfia para el ejercicio3.html
- html: 4 archivos .html con cada ejercicio
- js: 4 archivos .js que son utilizados en cada ejercicio

## ¿En qué consiste?
### El ejercicio1: 
Se codifica un texto dando la vuelta a todas las palabras que tenga este.
### El ejercicio2: 
Programa de lotería
### El ejercicio3: 
Se muestra la hora actual y el tiempo restante hasta el final de curso. Se actualiza cada segundo, viendo los segundos contar.
### El ejercicio4: 
Aparece el tamaño actual de la ventana (ancho y alto) al pulsar un botón.

## Cómo funciona
Iniciar abriendo los archivos html para ver cada ejercicio
